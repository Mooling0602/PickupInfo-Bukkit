# PickupInfo

一个 Bukkit 插件，在玩家动作栏（ActionBar）显示物品变化信息（拾取、丢弃、指令清除）。

## 功能

- 拾取物品时显示**绿色** `+N 物品名`
- 丢弃或通过 `/clear` 清除物品时显示**红色** `-N 物品名`
- 同一 tick 内的多次变化合并为一条动作栏消息
- 无需配置文件，丢入 plugins 即可使用

## 环境要求

- Paper 1.20.4+（或兼容 Adventure API 的服务端）
- JDK 17+

## 构建

```sh
mvn clean package
```

编译产物为 `target/PickupInfo-1.0.0.jar`。

## 使用方法

1. 将 `PickupInfo-1.0.0.jar` 放入服务端 `plugins/` 目录
2. 重启服务器或执行 `/reload`
3. 正常拾取/丢弃物品即可 — 动作栏会自动显示变化

### 消息示例

| 操作 | 动作栏显示 |
|------|-----------|
| 拾取 2 个钻石 | `+2 钻石`（`+2` 为绿色） |
| 丢弃 1 个圆石 | `-1 圆石`（`-1` 为红色） |
| 执行 `/clear` | `-15 泥土 \| -3 橡木原木`（均为红色） |
| 同时拾取多种物品 | `+5 小麦 \| +3 骨头` |
